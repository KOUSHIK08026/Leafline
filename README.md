# Leafline

Leafline is a fully functional, project-based task tracker web application built as a single HTML file. It uses a clean, cyan-themed "Crystalline Growth" design system to help organize and visualize tasks.

## Features
- **Project & Task Management**: Create multiple projects and organize tasks under them.
- **Tree Structure**: Interactive task tree with nested child tasks.
- **Drag & Drop**: Seamlessly reorganize tasks.
- **Data Persistence**: Uses browser `localStorage` to save your projects automatically.
- **Modern UI**: Polished hover-based actions and modern, responsive design.
